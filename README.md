# C_Dentist_VR_AR_
#8. 어린이용 치과의사 체험_VR_AR

- 시현 동영상(모바일, PC), 스크립트, .apk파일 첨부.

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

## 1. 개발 계기

```
2020년 4학년 2학기 장기인턴으로 마지막 학기 진행.
VR(모바일) Unity로 개인 개발.
어린이들이 직접 손으로 움직이고 체험할 수 있는 것을 생각했다.
Cardboard를 끼고 몰입감을 높이며 마커를 통해 직접 움직임까지 더하면
실감나게 직접 체험할 수 있다는 생각으로 개발하게 되었다.
```
